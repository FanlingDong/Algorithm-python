## 迭代法
### 前序pre：中左右
> 栈先进后出
> 先把cur压栈，再弹出cur，将他的左右节点压栈，注意先压右孩子，再压左孩子

#### 伪代码
```python
def traversal(root):
	if not root:
		return None
	stack = [root]
	res = []
	while stack:
		cur = stack.pop()
		if cur != None:
			res.append(cur.val)
		else:
			continue
		stack.append(cur.right)
		stack.append(cur.left)
	return res
```

###  后序遍历post：左右中
前序中左右 -> 将左右子节点入栈顺序颠倒变成 中右左
再对res做翻转，就是左右中
#### 伪代码
```python
def traversal(root):
	if not root:
		return None
	stack = [root]
	res = []
	while stack:
		cur = stack.pop()
		if cur != None:
			res.append(cur.val)
		else:
			continue
		stack.append(cur.left)
		stack.append(cur.right)
	res.reverse()
	# 或者用切片：res = res[::-1]
	return res
```
